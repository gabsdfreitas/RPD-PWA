Registro de Pensamentos Disfuncionais

Uma ferramenta web interativa e moderna para auxiliar no processo de Terapia Cognitivo-Comportamental (TCC), permitindo identificar, registrar e reestruturar pensamentos automáticos disfuncionais.

Como Usar
Instalação
Não há instalação necessária! Este é um aplicativo web puro (HTML + CSS).

Clone o repositório:

git clone https://github.com/alteralph/RPD-PWA

Abra o arquivo HTML:

Navegue até a pasta do projeto
Abra registro-pensamentos.html em qualquer navegador moderno
Pronto! O aplicativo já está funcionando
